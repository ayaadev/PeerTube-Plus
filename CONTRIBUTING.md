# Contributing

## Reporting a bug
- **Do not open a GitHub issue if the bug is a security vulnerability.** Instead, refer to the [SECURITY.md](/SECURITY.md) file.
- Please check if the bug already has an open issue in the [Issues](https://github.com/ayaadev/PeerTube-Plus/issues) tab. If there is already an open issue, please contribute to that instead.
- If there is not an open issue for the bug, please [create a new issue](https://github.com/ayaadev/PeerTube-Plus/issues/new/choose).

## Adding/modifying code (pull requests)
- Please [open a new pull request](https://github.com/ayaadev/PeerTube-Plus/compare).
- Ensure the pull request has a clear description which clearly describes:
  - **If you are fixing a bug**: The issue you are fixing and how you fixed it. If there is a relevant issue to your pull request, please include the issue number.
  - **If you are adding a new feature**: Why you believe the feature should be added and what changes you made to the code.

## General discussions (questions, etc)
Please [open a discussion](https://github.com/ayaadev/PeerTube-Plus/discussions/new/choose) in the relevant category.

Thank you very much for contributing!
